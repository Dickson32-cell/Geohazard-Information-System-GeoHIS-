# Project Overview

## Project Name
Tunchee WebPage Demo

## Description
A demonstration webpage for the Tunchee project, showcasing web development capabilities and features.

## Tech Stack

### Frontend
- HTML5
- CSS3
- JavaScript (ES6+)

### Backend
- None (static webpage)

### Development Tools
- VS Code
- Git

### Deployment
- Static hosting (GitHub Pages, Netlify, etc.)

## Coding Conventions

### HTML
- Use semantic HTML5 elements
- Lowercase tag names
- Double quotes for attributes
- Proper indentation (2 spaces)

### CSS
- Use CSS3 features where appropriate
- BEM methodology for class naming
- Mobile-first responsive design
- CSS custom properties (variables) for theming

### JavaScript
- ES6+ syntax (const/let, arrow functions, etc.)
- Modular code structure
- Event delegation where appropriate
- Avoid global variables

### File Naming
- Lowercase with hyphens for multi-word names
- Descriptive names that reflect content/purpose

### Version Control
- Git for version control
- Descriptive commit messages
- Feature branches for new development

## Development Workflow
1. Create feature branch
2. Implement changes
3. Test in multiple browsers
4. Commit with descriptive message
5. Push to remote repository
6. Create pull request for review
